# Famous Quotes API

偉人の名言を取得できるシンプルな Web API です。  
Express（Node.js）を用いて作成しており、**WebAPIの基礎理解を目的**とした学習用プロジェクトです。
フロントエンド（React / Flutter など）からの利用を前提とし、  
API は **データ提供の責務のみに限定**しています。

---

## 概要

- 偉人の名言データを JSON 形式で保持
- クライアントから指定された `id` に対応する名言を返却
- ランダム処理はフロントエンド側で行う設計
- CRUD のうち **Read（取得）に特化**

---

## 技術スタック

- Node.js
- Express
- JavaScript

## データ設計

名言ー他は、サーバ起動中のみに有効になるインメモリ配列として管理。GETメソッドの時に、json化してレスポンスを返す。
```json
{
  "index": 0,
  "quote": "もし今日が人生最後の日だとしたら、今やろうとしていることは本当に自分のやりたいことだろうか？",
  "author": {
    "name": "スティーブ・ジョブズ",
    "overview": "アップル社の創業者。革新的な製品を世に送り出した",
    "born": 1955,
    "died": 2011
  }
}
```
### 設計方針
・index
    名言を一意に認識するid
・author
    著者名をオブジェクトにし、拡張可能な構造にしてた
・died
    存命中の人物は現在の西暦を設定している

## API設計

### 名言の取得
指定したIDに対応する名言を取得する
```text
GET /quotes/:id
```
リクエスト例
```text
GET /quotes/3
```
成功時は以下のレスポンスを返す
```json
{
  "index": 3,
  "quote": "もうこれで満足だという時は、すなわち衰える時である。",
  "author": {
    "name": "渋沢栄一",
    "overview": "日本の武士、官僚、実業家、日本資本主義の父",
    "born": 1840,
    "died": 1931
  }
}
```
失敗時は以下のレスポンスを返す
```json
{
  "message": "指定されたIDの名言は存在しません"
}
```
